# KnowledgeForge

AI-powered knowledge management system that transforms YouTube videos into structured notes using multi-agent workflows.

## Tech Stack

- FastAPI
- LangGraph
- LangChain
- Ollama
- OpenAI
- Qdrant
- Notion

## Roadmap

### Version 1.0 (Current Goal)

- ✅ Provider-agnostic LLM Service
- ✅ Knowledge Agent
- ✅ Health API
- 🚧 KnowledgeDocument
- ⬜ Reflection Agent
- ⬜ YouTube Transcript Extraction
- ⬜ Markdown Export
- ⬜ End-to-End YouTube → Notes Workflow
